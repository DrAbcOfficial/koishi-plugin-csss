# koishi-plugin-csss

[![npm](https://img.shields.io/npm/v/koishi-plugin-csss?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-csss)

cs server status

## 提供指令
- ```cs [地址:端口] <-i | -t | -c>```

> [ ]: 必填 < >: 选填  
> -i 生成图片横幅  
> -t 输出文本信息  
> -c 清除缓存

- `csss`

> 选项:
> -i 生成图片横幅  
> -t 输出文本信息  
> -c 清除缓存
